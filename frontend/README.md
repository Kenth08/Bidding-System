# Blockchain E-Procurement System

Professional React landing page for a blockchain-based e-procurement capstone project.

## Tech Stack

- React
- Vite
- Tailwind CSS
- lucide-react

## Run Locally

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Build for production:

   ```bash
   npm run build
   ```

## Project Structure

- `src/main.jsx` boots the React app
- `src/App.jsx` renders the landing page
- `src/components/LandingPage.jsx` contains the full UI
