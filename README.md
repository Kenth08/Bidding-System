# Blockchain E-Procurement System

A blockchain-based procurement bidding system.
Built for BSIT Capstone 2026 — Davao del Norte State College.

## Tech Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Django REST Framework
- Database: PostgreSQL (local) / Supabase (production)

## Setup

### Backend
```bash
cd backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
